3D Flocking simulation using GPGPU programming (OpenGL, GLSL, C++) Github

Dissertation project. Implemented a standard flocking algorithm on the GPU, later optimised the nearest neighbour search using spatial binning, compared and tested performance. The project aim was to investigate the GPU capabilities using OpenGL Compute Shaders for general particle simulations. Learned about advantages and limitations of GPU compute, implemented a parallel prefix scan and counting sort.
