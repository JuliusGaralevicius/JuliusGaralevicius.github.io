An android app project which was developed during second year in University.

With help of other students, created an android application for a city council which features collaborative issue reporting. The main idea of the app is to allow the council are residents to report issues about the community in the app, tag them on the map (for example a broken bench at some location) and then, whoever controls the database or has access to application can check and resolve the issues. Features that I have worked on exclusively include:

Issue reporting using Firebase real-time database and Google Maps Api


Screenshot database "resolved" property, change it's value. Marker turns green.
Screenshot of markers database refering users and photos

User ranking and rewards


To partially prevent the map from being spammed with markers which have nothing to do with issues (selfies, random photos), I have implemented a user ranking system. Each user starts with 1 star and gets 1 report per day. Users get points from logging in and having their issues resolved. As users get more points, they gain extra stars and raise more issues.
